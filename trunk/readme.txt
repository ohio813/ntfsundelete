How to Compile
--------------

1. Install Virtual Tree View
    http://www.delphi-gems.com/index.php?option=com_content&task=view&id=12&Itemid=38

2. Install Python For Delphi
    http://membres.lycos.fr/marat/delphi/python.htm

2.1. Patch Python For Delphi to support Python 2.4
    (patch in the "PythonForDelphiPatch" folder)
    Support for later python releases can be added similary (view the contents of the PythonEngine.pas)

3. Install Python

4. Install VFSManager
    (Sources/vfsmgr/vfsmgr.dpk)
